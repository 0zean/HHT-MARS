# HHT-MARS
A hybrid forecasting model that combines the Hilbert-Huang Transform and Multivariate Adaptive Regression Splines

**Requirements**
- numpy
- pandas/pandas-datareader
- pyhht
- sklearn
- pyearth
- xgboost
- datetime
- matplotlib
